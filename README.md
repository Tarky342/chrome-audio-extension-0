

---

### **インストール手順**
1. **Chrome の拡張機能管理ページを開く**  
   Chromeのアドレスバーに以下を入力し、Enterキーを押す。  
   ```
   chrome://extensions/
   ```
2. **デベロッパーモードをONにする**  
   画面右上にある「デベロッパーモード」のスイッチをONにする。
3. **拡張機能を読み込む**  
   - 「パッケージ化されていない拡張機能を読み込む」ボタンをクリック。  
   - 拡張機能のフォルダを選択する。

---

### **使い方**
#### **Chrome拡張のUIから操作**
1. **拡張機能のアイコンをクリック**  
   Chromeのツールバーに表示されている拡張機能アイコンをクリック。
2. **オーディオ出力デバイスを選択**  
   - リストからオーディオ出力デバイスを選択する。
   - **現在はOS上の仮想デバイスのみ対応（既定のデバイスなど）**。
3. **「適用」ボタンを押す**  
   選択したデバイスにオーディオ出力を適用。
